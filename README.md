API (публичные функции)
----------------------
- get_mask_card_number(value: Union[str, int]) -> str
  - Описание: маскирует номера карт, сохраняя форматирование (пробелы, дефисы и др.). В большинстве случаев видимы последние 4 цифры, остальное заменяется символами маски.
  - Вход: строка с произвольными символами или целое число. Если строка не содержит цифр — ожидаемое поведение: вернуть исходную строку (см. тесты).

- get_mask_account(value: Union[str, int]) -> str
  - Описание: маскирует номер аккаунта/счёта. Возвращает строку вида `"**<visible>"`, где `<visible>` — последние 1–4 цифры в зависимости от длины. Если цифр нет — возвращает `"**"`.
  - Вход: строка или целое число. Минус в отрицательных числах игнорируется (берутся только цифры).

Примеры использования
---------------------
```python
from src.masks import get_mask_card_number, get_mask_account

print(get_mask_card_number("4111 1111 1111 1111"))  # => маскированная строка
print(get_mask_account(123456789012))                 # => "**9012"
```

Правила поведения (из тестов)
----------------------------
- get_mask_card_number:
  - Если в строке не найдено ни одной цифры — возвращается исходная строка.
  - Строки с разделителями (пробелы, дефисы и т.д.) сохраняют позиционные маркеры, маскируются только цифры по правилу.
  - Короткие наборы цифр (<4) не маскируются согласно тестовому сценарию.

- get_mask_account:
  - При отсутствии цифр возвращает `"**"`.
  - Для чисел и строк с цифрами оставляет видимыми последние 1–4 цифры и прячет остальное под `*`.
